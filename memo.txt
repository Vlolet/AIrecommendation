옷:
    부위(모자/상의/하의/신발/가방/액세서리)
    색상
    스타일

data:
    옷
    옷 조합 - 성별/퍼스널컬러/상황/스타일/날씨

input:
    유저 정보
        (성별, 퍼스널 컬러, 보유 옷 등)
    상황(경조사, 데이트, 출근 등)
    스타일(귀여운, 캐주얼, 펑크 등)
    날씨(온도/눈-비-바람 등)

output:
    옷 조합




옷 정보에 들어갈 거
계절 / 카테고리(e.g. 상의 -> 니트) / 색상 / 소재 / 패턴

옷 dataset
    옷의 정보가 들어감

패션(옷의 조합) dataset
    조합된 옷의 정보 + 이 조합이 의미하는 바(여자/가을/캐주얼)


 옷 인식 모델(패션추천모델 전처리 & 이미지 업로드 용도)
 Classification : 사물 하나의 종류를 분류
 Object Detection : 다양한 사물이 있을 때 경계박스로 위치와 종류를 분류
